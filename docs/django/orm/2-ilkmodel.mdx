## Modellerin oluşturulması

Öncelikle `inventory/models.py` içine her bir tablo için bir sınıf açarız. Burada şimdilik product ve brand olarak iki sınıf var

```python title="inventory/models.py" showLineNumbers
from django.db import models


class Product(models.Model):
    name = models.CharField(max_length=100)
    qty = models.IntegerField()


class Brand(models.Model):
    name = models.CharField(max_length=100)
```

Veri tabanımıza (`db.sqlite3`) baktığımızda inventory_brand ve inventory_product olarak iki farklı tablo oluşturulduğunu göreceğiz. Bu tablolarda açtığımız sütunlara ek olarak bir de id sütunu bulunacaktır. Bu varsayılandır ve otomatik olarak eklenir.

Burada sınıflar altına bir string representation açmamız uygun olacaktır. Normal şartlarda sınıfları çağırdığımızda sınıflar çıktı olarak nesnelerin veya sınıfların hafıza adreslerini verir. Bunu, sınıf içine `def __str__(self)` şeklinde bir fonksiyon ekleyerek ve sınıf içindeki nesnelerden birisini return ederek kullanıcı dostu ve daha okunabilir bir forma dönüştürebiliriz. Yani yeni kodumuz şöyle olacaktır:


```python title="inventory/models.py" showLineNumbers
from django.db import models


class Product(models.Model):
    name = models.CharField(max_length=100)
    qty = models.IntegerField()

    def __str__(self):
        return self.name


class Brand(models.Model):
    name = models.CharField(max_length=100)
```
