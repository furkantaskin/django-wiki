# Yeni Notlar

Bu kısımdan itibaren artık temel Django kısımlarını bir kenara bırakıp ileride takıldığım yerleri çözebilmek amacıyla notlar alacağım. Örnek olarak anket projesinde static dosyaları çekerken birkaç sorun yaşadım. Bunları tam anlamaıyla anlamam gerekiyor. Bu yüzden [HowTo](https://docs.djangoproject.com/en/4.1/howto/) sayfasını takip ederek temel Django olaylarını iyice kavramak ve kendimi bu kısımda ilerletmek istiyorum.
